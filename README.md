## Quickstart

### Install

```sh
git clone https://github.com/ctate/codeless.git
cd codeless
npm install
```

### Configure

Add `.env.local` file with your OpenAI [API Key](https://platform.openai.com/account/api-keys):

```
OPENAI_API_KEY=""
```

### Run

```
npm run dev
```

Then go to: http://localhost:62223
